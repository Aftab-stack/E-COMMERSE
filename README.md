# React E-Commerce Application

A fully functional and responsive e-commerce application built with **React**, designed as a capstone project to showcase frontend skills.

## Features

- **Product Listing**: Display products fetched from FakeStore API.
- **Product Details**: View detailed product information.
- **Shopping Cart**: Add/remove products with persistent storage using Context API.
- **User Authentication Simulation**: Login/Logout flow using localStorage.
- **Checkout Process**: Form validation for placing orders.
- **Responsive Design**: Works on desktop and mobile devices.
- **Performance Optimizations**: Lazy loading and code splitting.
- **Modern UI**: Clean, attractive layout with improved spacing, typography, and colors.

## Technologies Used

- React (Functional Components + Hooks)
- React Router (Navigation)
- Context API (State Management)
- FakeStore API (Data Source)
- CSS (Styling and Responsive Design)
- Vite (Project Bundler)
